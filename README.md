# termcolors
terminal colors libary for C/C++


# usage
if on windows:
```cpp
init_ansi();
std::printf(GREEN("hi"));
```
or
```c
init_ansi();
printf(GREEN("hi"));
```
if on linux
```cpp
std::printf(GREEN("hi"));
```
or
```c
printf(GREEN("hi"));
```
